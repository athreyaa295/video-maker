# Project Management - Full Stack GPU HMX1008

## Goal
Automate 60-second highlight reel generation using GPU and provide a **React frontend** for user interaction.

## Milestones
1. Setup backend environment (Python, FastAPI, Ollama, FFmpeg)
2. Implement video + audio extraction
3. GPU Whisper transcription
4. GPU LLM highlight scoring
5. Video stitching
6. Setup React frontend with SCSS/Tailwind
7. Connect frontend → backend API
8. Testing & validation
9. Documentation
